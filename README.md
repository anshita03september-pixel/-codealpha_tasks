# 🌐 Personal Portfolio Website

## 📌 Project Overview

This project is a modern and responsive **Personal Portfolio Website** developed using **HTML5, CSS3, and JavaScript** as part of the **CodeAlpha Frontend Development Internship – Task 3**.

The website showcases my personal profile, technical skills, projects, resume, and contact information in a clean and interactive layout.

# ✨ Features

- Responsive Design
- Modern User Interface
- Smooth Scrolling Navigation
- Sticky Navigation Bar
- Animated Home Section
- About Me Section
- Skills Section
- Projects Showcase
- Resume Download Button
- Contact Form
- Social Media Icons
- Scroll-to-Top Button
- Typing Animation

# 🚀 How to Run the Project

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in any web browser.

OR

1. Open the folder in **Visual Studio Code**.
2. Install the **Live Server** extension.
3. Right-click **index.html**.
4. Click **Open with Live Server**.
